# matheusode.me

Este é o meu site pessoal. Aqui você pode ver meus projetos, um pouco sobre mim e minha experiência, e minhas informações de contato (sinta-se à vontade para entrar em contato :)).

## Do que é feito?

React, TypeScript e Vite.

## Como usar?

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/MatheusOde/matheusode.me.git
    ```

2.  **Instale as dependências:**

    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**

    ```bash
    npm run dev
    ```

## Como fazer o deploy?

O deploy é feito usando o script `deploy.sh`.

1.  **Dê permissão de execução para o script:**

    ```bash
    chmod +x deploy.sh
    ```

2.  **Execute o script com uma mensagem de commit:**

    ```bash
    ./deploy.sh "Sua mensagem de commit"
    ```
